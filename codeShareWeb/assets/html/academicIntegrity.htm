CodeShareWeb.com Academic Integrity

Academic Integrity Agreement

1.       Definitions

1.1    Academic honesty means the use of one's own thoughts and materials in the writing of papers, taking of tests, and other classroom related activities. Students intentionally aiding other students in any infraction of the academic honesty policy are considered equally guilty.

1.2    Academic integrity is the moral code or ethical policy of academia. This includes values such as avoidance of cheating or plagiarism; maintenance of academic standards; honesty and rigor in research and academic publishing.

1.3    Plagiarism is the practice of taking someone else's work or ideas and passing them off as one's own.

1.4    Open-source software (OSS) is computer software with its source code made available with a license in which the copyright holder provides the rights to study, change, and distribute the software to anyone and for any purpose. Open-source software may be developed in a collaborative public manner.

1.5    Intellectual Property refers to a work or invention that is the result of creativity, such as a manuscript or a design, to which one has rights and for which one may apply for a patent, copyright, trademark, etc.

2.       Open-Source Policy

2.1    CodeShareWeb.com makes source code available open-source.  The code samples may be utilized for future development.  The open-source format does not grant the user permission to represent the work as their own intellectual property for either profit or educational products.

2.2    Developers utilizing open-source code samples from the CodeShareWeb.com repository must attribute credit to CodeShareWeb.com and the original author in the documentation of the source code utilizing the sample.

2.3    Students may utilize the code samples in the CodeShareWeb.com repository for academic purposes, providing that authorship is attributed to CodeShareWeb.com and the original author, and that the use of code for assistance in development is not prohibited by the University or individual course professor.

3.       Academic Permissions

3.1    The purpose of the code samples provided by CodeShareWeb.com is primarily educational.

3.2    You are subject to your institutions code of student conduct while using CodeShareWeb.com.  CodeShareWeb.com will not be held accountable for student violations of institutional conduct including but not limited to:

(a)    inadvertent use of code samples in courses where the use is not permissible. 

(b)    failure to properly cite code samples and inspiration from CodeShareWeb.com leading to disciplinary action.

(c)    disciplinary actions as a result of over use of assistance or creation of derivative code not sufficiently unique.

3.3     Users of CodeShareWeb.com are responsible for ensuring they do not violate any institutional policies in the use of CodeShareWeb.com.

4.       Sharing Code Samples

4.1    CodeShareWeb.com allows for students to share their original works to provide guidance and instructional support.  You may:

(a)    share code samples that you have personally written;

(b)    share code samples written by a group providing that all contributors are given credit;

(c)    share code samples that contain open-source code providing that this is a permissible use under the open-source license;

(e)    share comments, suggestions or other information that is informational in nature related to the creation of code:

(f)    share documentation or supplementary files necessary for the use of code samples,

        subject to the other provisions of these terms and conditions.

4.2    Your code will be removed from the database if it violates the spirit of academia including, but not limited to:

        (a)       revealing information that compromises or undermines a professors ability to provide instruction in a subject;

        (b)       discloses confidential test or assignment questions/answer;

        (c)       shares copyrighted information of any kind,

        at the request of individual professors, university staff or copyright holders.
